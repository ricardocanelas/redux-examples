# Redux Examples

Until the moment there are just two examples.

## 👉🏻 Prerequisites

* Git
* Node
* NPM

## 👉🏻 Installing

1. Clone this repository using `git clone`
2. Onto the repository, install the dependecies using `npm install` or `yarn install`
